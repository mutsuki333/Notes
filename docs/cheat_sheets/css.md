### Image align vertically and horizontally

```sass
.container
  position: relative
  height: 40vh // can be anything
  img
    max-height: 100%
    max-width: 100%
    width: auto
    height: auto
    position: absolute
    top: 0
    bottom: 0
    left: 0
    right: 0
    margin: auto
```
